# Student Management System
A full-stack Java Spring Boot application for managing student records.

## Features
* **CRUD**: Create, Read, Update, and Delete students.
* **Search**: Filter students by name.
* **Reports**: Real-time calculation of total students and class average marks.
* **Database**: Uses H2 In-Memory database (No installation required).

## Tech Stack
* Java 17+
* Spring Boot 3
* Thymeleaf (Frontend)
* Bootstrap 5 (Styling)
* Maven

## How to Run
1. Open in VS Code.
2. Ensure Java Extension Pack is installed.
3. Run `ManagementApplication.java`.
4. Visit `http://localhost:8080`.
